# Rocket Rascals: Snack Attack! 🚀

A whimsical full-screen space shooter — fly a rocket and fling silly things (gummy worms, rubber duckies, cream pies, googly eyes, spiders, donuts...) at cuddly space critters drifting down toward your ship.

## Play
Open the GitHub Pages link, or open `index.html` in any browser.

## Controls
- **Arrow keys** — fly your rocket
- **Space** — shoot (hold for rapid fire)
- **F** — fullscreen
- **P** — pause

Kid-friendly, unlimited ammo. Built for a 7–8 year old.
